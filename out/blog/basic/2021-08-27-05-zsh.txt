b:T977,<p>리눅스에서는 터미널을 많이 사용한다. 이 때 명령을 해석하고 실행시키는 것이 <code>shell</code>이다. 우분투에서는 기본 쉘로 <code>bash</code>를 사용한다.</p>
<p><code>zsh</code>는 <code>bash</code>보다 기능도 많고 플러그인이나 테마를 이용해 커스터마이징이 가능해서 기본 쉘보다 더 예쁘고 편리하게 터미널을 사용할 수 있도록 해준다. 이를 위해서는 <code>oh-my-zsh</code>도 같이 설치해야 한다.</p>
<blockquote>
<p><a href="/linux/06-oh-my-zsh/">oh-my-zsh 설치</a></p>
</blockquote>
<p></p>
<h1>1. zsh 설치</h1>
<p>현재 사용 중인 쉘을 확인해보자.</p>
<pre><code class="language-zsh">$ echo $SHELL
</code></pre>
<pre><code>/bin/bash
</code></pre>
<p>기본 쉘은 <code>bash</code>이다.</p>
<p>이제 <code>zsh</code>를 설치한다.</p>
<pre><code class="language-zsh">$ sudo apt-get install zsh
</code></pre>
<hr>
<h1>2. zsh를 기본 쉘로 변경</h1>
<p><code>zsh</code>가 설치된 위치를 확인한다.</p>
<pre><code class="language-zsh">$ which zsh
</code></pre>
<pre><code>/usr/bin/zsh
</code></pre>
<p>확인한 위치를 넣어 기본 쉘을 변경한다.</p>
<pre><code class="language-zsh">$ chsh -s /usr/bin/zsh
</code></pre>
<p>그러고 터미널을 다시 시작하니 이런 화면이 떴다.</p>
<p><img src="https://user-images.githubusercontent.com/86853786/131117079-ccbb4657-6901-48de-ad4f-727e7249808a.png" alt="image"></p>
<ul>
<li><strong>(q)</strong> : 그냥 종료한다. 다음에 이 화면이 다시 뜬다.</li>
<li><strong>(0)</strong> : <code>~/.zshrc</code> 파일을 생성만 하고 종료한다. 이 화면은 다시 뜨지 않는다.</li>
<li><strong>(1)</strong> : 메인 메뉴로 들어간다.</li>
<li><strong>(2)</strong> : <code>~/.zshrc</code> 파일을 생성하고 추천하는 기본값으로 세팅한다.</li>
</ul>
<p>나는 (2)를 선택했다. 쉘이 이렇게 바뀌었다.</p>
<p><img src="https://user-images.githubusercontent.com/86853786/131118358-b13eae06-bea1-4ca7-b6bc-4bfc5d748422.png" alt="image"></p>
<p>다시 사용 중인 쉘을 확인해보면 <code>zsh</code>가 잘 나온다.</p>
<pre><code class="language-zsh">$ echo $SHELL
</code></pre>
<pre><code>/usr/bin/zsh
</code></pre>
