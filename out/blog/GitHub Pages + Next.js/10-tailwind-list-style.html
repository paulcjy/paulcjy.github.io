tailwind css에서 리스트 항목(<li>)에 숫자나 기호 표시하기

2023. 11. 20. 09:00

tailwind css에서 리스트를 사용하면 앞에 목록 기호가 나타나지 않는다. <ul>과 <ol> 둘 다 그렇다. globals.css에서 아래 코드를 추가하면 원래대로 돌아온다.

@tailwind base;
@tailwind components;
@tailwind utilities;

@layer base {
  ul,
  ol {
    list-style: revert;
  }
}
