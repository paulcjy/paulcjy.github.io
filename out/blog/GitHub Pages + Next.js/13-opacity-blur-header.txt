b:T8ac,<h1>투명도(opacity)</h1>
<p><a href="https://developer.mozilla.org/en-US/docs/Web/CSS/opacity" title="mdn opacity"><code>opacity</code></a>는 0에서 1 사이의 값을 가진다. opacity는 불투명도라는 뜻이라서 0은 완전 투명, 1은 완전 불투명을 의미한다. CSS에서는 <code>opacity: 0</code>의 형식으로 지정할 수 있다. 숫자도 가능하고 퍼센트(%)도 가능하다.</p>
<pre><code class="hljs language-css"><span class="hljs-attribute">opacity</span>: <span class="hljs-number">0.9</span>;
<span class="hljs-attribute">opacity</span>: <span class="hljs-number">90%</span>;
</code></pre>
<p><strong>tailwind</strong>에서는 색상 뒤에 <code>/</code>를 쓰고 0~100의 값을 입력하면 된다.</p>
<pre><code class="hljs language-html"><span class="hljs-comment">&#x3C;!-- 완전 투명 --></span>

</code></pre>
<h1>블러(blur)</h1>
<p>블러를 줄 때는 <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/backdrop-filter" title="mdn backdrop-filter"><code>backdrop-filter</code></a> 속성을 사용한다. 필터의 종류는 굉장히 많다.</p>
<ul>
<li>blur</li>
<li>brightness</li>
<li>contrast</li>
<li>drop-shadow</li>
<li>grayscale</li>
<li>hue-rotate</li>
<li>invert</li>
<li>opacity</li>
<li>sepia</li>
<li>saturate</li>
</ul>
