<h1>1. 제목 Headings</h1>
<p>제목은 1단계부터 6단계까지 있다. 각각 HTML의 <code>&#x3C;h1></code>~<code>&#x3C;h6></code>에 해당한다. 제목을 표시하기 위해서는 <code>#</code>을 앞에 붙이면 된다.</p>
<pre><code># 제목1
## 제목2
### 제목3
#### 제목4
##### 제목5
###### 제목6
</code></pre>
<p></p>
<h1>제목1</h1>
<h2>제목2</h2>
<h3>제목3</h3>
<h4>제목4</h4>
<h5>제목5</h5>
<h6>제목6</h6>
<p></p>
<p>제목1과 제목2는 다른 방법으로도 표시할 수 있다. 제목1은 <code>===</code>를 사용하고 제목2는 <code>---</code>를 사용한다. 각 기호를 3개 이상 사용하면 제목으로 인식된다.</p>
<p><strong>제목1</strong></p>
<pre><code>제목1
===
</code></pre>
<p><strong>제목2</strong></p>
<pre><code>제목2
---
</code></pre>
<hr>
<h1>2. 문단 Paragraphs</h1>
<p>문단을 나누기 위해서는 빈 줄을 한 개 이상 삽입하면 된다. HTML의 <code>&#x3C;p></code>에 해당한다. 여러 번 줄바꿈을 해도 하나로 인식된다.</p>
<pre><code>첫번째 문단입니다.

두번째 문단입니다.



세번째 문단입니다.
</code></pre>
<p>첫번째 문단입니다.</p>
<p>두번째 문단입니다.</p>
<p>세번째 문단입니다.</p>
<hr>
<h1>3. 줄바꿈 Line Breaks</h1>
<p>마크다운에서는 엔터를 친다고 해서 줄바꿈이 이루어지지 않는다. 줄바꿈을 위해서는 문장 맨 끝에서 <strong>공백 2칸</strong> 이상을 띄고 엔터를 쳐야 한다. <code>&#x3C;br></code>을 직접 입력해도 줄바꿈이 된다.</p>
<pre><code>첫번째 문장
입니다.··
두번째 문장
입니다.
</code></pre>
<p>첫번째 문장입니다.<br>
두번째 문장입니다.</p>
<pre><code>첫번째 문장
입니다.&#x3C;br>
두번째 문장
입니다.
</code></pre>
<p>첫번째 문장입니다.
두번째 문장입니다.</p>
<hr>
<h1>4. 구분선 Horizontal Rules</h1>
<p>구분선을 긋고 싶을 때는 별표(***), 하이픈(---), 밑줄(___)을 3개 이상 사용하면 된다. HTML의 <code>&#x3C;hr></code>에 해당한다.</p>
<pre><code>***
---
___
</code></pre>
<p>단, 하이픈을 사용할 때는 윗줄이 비어있어야 한다. 윗줄에 텍스트가 있다면 <code>&#x3C;h2></code>로 인식하여 제목이 된다.</p>
<pre><code>잘못된 예시
-----------
</code></pre>
<hr>
<h1>5. 강조 Emphasis</h1>
<p>강조에는 <strong>굵게</strong>, <strong>이탤릭체</strong>, <strong>취소선</strong>이 있다. 각각 HTML의 <code>&#x3C;strong></code>, <code>&#x3C;em></code>, <code>&#x3C;del></code>에 해당한다. 굵게와 이탤릭체는 텍스트를 <code>*</code>와 <code>_</code>로 감싸주는데, 굵게는 2개, 이탤릭체는 1개로 감싸준다. 취소선은 <code>~~</code>로 감싸준다.</p>
<pre><code>**굵은 글씨 1**
__굵은 글씨 2__
*이탤릭체 1*
_이탤릭체 2_
~~취소선~~
</code></pre>
<p><strong>굵은 글씨 1</strong><br>
<strong>굵은 글씨 2</strong><br>
<em>이탤릭체 1</em><br>
<em>이탤릭체 2</em><br>
~~취소선~~</p>
<p>추가로 마크다운은 아니지만 <strong>HTML 태그</strong>를 이용하여 디자인을 바꿀 수 있다. <code>&#x3C;u></code>로 밑줄을 긋거나, <code>&#x3C;span></code>을 사용하여 css를 적용할 수 있다.</p>
<pre><code>&#x3C;u>밑줄 예시&#x3C;/u>
&#x3C;span style="color:red">빨간 글씨 예시&#x3C;/span>
</code></pre>
<p>밑줄 예시<br>
빨간 글씨 예시</p>

<h1>6. 인용 Blockquotes</h1>
<p>인용을 만들기 위해서는 문단의 맨 앞에 <code>></code>를 붙이면 된다. <code>></code>만 붙은 마크다운이라고 생각하면 편하다.</p>

>

>
> ### 제목이나
>
> **굵게**나 *이탤릭체*
> - 목록도 사용가능하다.
</code></pre>

<h1>7. 목록 Lists</h1>
<p>목록에는 HTML의 <code>&#x3C;ol></code>에 해당하는 <strong>순서가 있는 목록</strong>과, <code>&#x3C;ul></code>에 해당하는 <strong>순서가 없는 목록</strong>이 있다.</p>
<h2>순서가 있는 목록</h2>
<p><code>1.</code>와 같은 형식을 맨 앞에 붙이면 순서가 있는 목록이 된다. 이는 HTML로 바꿔보면 <code>&#x3C;ol></code> 안에 <code>&#x3C;li></code>로 들어가기 때문에 입력한 숫자에 상관 없이 자동으로 숫자가 매겨진다.</p>
<pre><code>1. 첫번째
2. 두번째
3. 세번째
4. 네번째
</code></pre>
<ol>
<li>첫번째</li>
<li>두번째</li>
<li>세번째</li>
<li>네번째</li>
</ol>
<p>숫자를 이상하게 입력해도 상관 없다. 단, 목록의 시작은 반드시 1번이어야 한다.</p>
<pre><code>1. 첫번째
1. 두번째
9. 세번째
5. 네번째
</code></pre>
<ol>
<li>첫번째</li>
<li>두번째</li>
<li>세번째</li>
<li>네번째</li>
</ol>
<p>숫자 앞에 <strong>공백 3칸</strong> 이상을 띄면 중첩된 형태도 만들 수 있다.</p>
<pre><code>1. 첫번째
2. 두번째
3. 세번째
   1. 중첩 1
   2. 중첩 2
4. 네번째
</code></pre>
<ol>
<li>첫번째</li>
<li>두번째</li>
<li>세번째
<ol>
<li>중첩 1</li>
<li>중첩 2</li>
</ol>
</li>
<li>네번째</li>
</ol>
<h2>순서가 없는 목록</h2>
<p><code>-</code>, <code>*</code>, <code>+</code>를 맨 앞에 붙이면 순서가 없는 목록이 된다. 마찬가지로 HTML에서 <code>&#x3C;ul></code> 안에 <code>&#x3C;li></code>로 들어가게 된다.</p>
<pre><code>- 목록 1
- 목록 2
* 목록 3
* 목록 4
+ 목록 5
+ 목록 6
</code></pre>
<ul>

<li>목록 2</li>
</ul>
<ul>
<li>목록 3</li>
<li>목록 4</li>
</ul>
<ul>
<li>목록 5</li>
<li>목록 6</li>
</ul>
<p>여기서는 맨 앞에 <strong>공백 2칸</strong> 이상을 띄면 중첩된 형태를 만들 수 있다.</p>
<pre><code>- 목록 1
- 목록 2
  - 중첩 1
  - 중첩 2
- 목록 3
</code></pre>
<ul>

<li>목록 2
<ul>

<li>중첩 2</li>
</ul>
</li>
<li>목록 3</li>
</ul>
<p>마크다운에는 표준이 없기 때문에 목록 기호가 다를 때 어떻게 처리할 지가 정해져 있지 않다. 따라서 기호를 섞어 쓰지 않는 것이 좋다고 한다.</p>

<h1>8. 코드 Code</h1>
<p>코드를 표현하는 방법은 두 가지가 있다. <strong>인라인 코드</strong>와 <strong>코드 블럭</strong>이다.</p>
<h2>인라인 코드 Inline Code</h2>
<p>인라인 코드는 원하는 텍스트를 grave accent(<code>`</code>)로 감싸주면 된다. 1개와 2개일 때 동작한다.</p>
<pre><code>`한 개`로 감쌀 때
``두 개``로 감쌀 때
</code></pre>
<p><code>한 개</code>로 감쌀 때<br>
<code>두 개</code>로 감쌀 때</p>
<h2>코드 블럭 Code Block</h2>
<p>코드 블럭을 만드는 방법은 두 가지가 있다. Grave accent를 3개 이상 사용하는 것(<code>```</code>)과 <strong>공백 4칸</strong>이나 <strong>탭 1칸</strong>을 띄는 것이다.</p>
<h3>Grave accent 사용</h3>
<pre><code>```
3개 이상으로 블럭을 만들 때는

grave accent 안에 들어있는 모든 것이 블럭이 된다.
```
</code></pre>
<pre><code>3개 이상으로 블럭을 만들 때는

grave accent 안에 들어있는 모든 것이 블럭이 된다.
</code></pre>
<h3>들여쓰기 사용</h3>
<pre><code>    4칸 띄어쓰거나
    1칸 탭을 한다.
</code></pre>
<pre><code>4칸 띄어쓰거나
1칸 탭을 한다.
</code></pre>
<h3>코드 하이라이팅</h3>
<p>시작하는 <code>```</code>옆에 소문자로 언어 이름을 적으면 코드에 하이라이팅이 된다.</p>
<pre><code>```c
#include &#x3C;stdio.h>

int main(void) {
	printf("Hello world");

	return 0;
}
```
</code></pre>
<pre><code class="language-c">#include &#x3C;stdio.h>

<h1>9. 링크 Link</h1>
<p>링크는 HTML의 <code>&#x3C;a></code>에 해당한다. 방법은 3가지가 있다.</p>
<h2>1) 주소를 그대로 사용</h2>
<p><code>&#x3C; ></code> 안에 주소를 넣으면 주소가 텍스트로 나오고 링크가 생성된다. 이메일도 사용할 수 있다.</p>
<pre><code>&#x3C;https://paulcjy.github.io/>
&#x3C;paulcjy@naver.com>
</code></pre>
<p><a href="https://paulcjy.github.io/">https://paulcjy.github.io/</a><br>
<a href="mailto:paulcjy@naver.com">paulcjy@naver.com</a></p>
<blockquote>
<p>물론 <code>&#x3C; ></code> 없이 그냥 주소만 입력해도 링크가 자동으로 생성된다.</p>
</blockquote>
<p>링크가 없이 주소 텍스트만 표시하고 싶을 떄는 주소를 <code>`</code>로 감싸주면 된다.</p>
<pre><code>`https://paulcjy.github.io/`
</code></pre>

<h2>2) 링크에 이름 붙이기</h2>
<p>주소 대신에 이름을 붙여 링크를 만들 수도 있다.</p>
<pre><code>[이름](주소 "설명")
</code></pre>
<p>이름은 페이지에 나오는 텍스트, 주소는 링크의 주소, 설명은 마우스를 링크 위에 올리면 뜨는 툴팁 내용이다. 설명은 생략할 수 있다.</p>

</code></pre>

<h2>3) 주석 형태로 링크 만들기</h2>
<p>링크 주소를 주석 형태로 만들면 마크다운 파일의 가독성이 좋아진다. 본문에는 링크 이름을 넣고 다른 곳에 주소를 적어 연결시키는 방식이다.</p>
<p>본문에 들어가는 부분은 <strong>대괄호 2개</strong>로 작성한다.</p>
<pre><code>[이름][라벨]
</code></pre>
<p>주소를 적을 때는 반드시 <strong>대괄호</strong> 안에 <strong>라벨</strong>을 적고 바로 <strong>콜론</strong>을 쓴 뒤 <strong>한 칸</strong>을 띄어야 한다. 그러고 주소와 설명을 이어 적는다. 주소는 그냥 쓰거나 <code>&#x3C; ></code>로 묶어도 된다. 라벨은 숫자, 문자, 공백이 모두 사용 가능하다. 설명은 큰 따옴표나 작은 따옴표로 묶을 수 있다. 효과는 모두 동일하다. 역시 설명은 생략할 수 있다.</p>
<pre><code>[라벨]: 주소
[라벨]: 주소 "설명"
[라벨]: 주소 '설명'
[라벨]: &#x3C;주소> "설명"
[라벨]: &#x3C;주소> '설명'
</code></pre>
<p><strong>예시</strong></p>
<pre><code>제 [블로그][1]로 놀러오세요.
[네이버 바로가기][네이버]

[1]: https://paulcjy.github.io/ "paulcjy blog"
[네이버]: https://www.naver.com/ 'naver'
</code></pre>
<p>제 <a href="https://paulcjy.github.io/" title="paulcjy blog">블로그</a>로 놀러오세요.<br>
<a href="https://www.naver.com/" title="naver">네이버 바로가기</a></p>

<h1>12. 표 Table</h1>

| text21 | text22 | text23 |
| text31 | text32 | text33 |
</code></pre>
