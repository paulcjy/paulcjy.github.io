b:T60a,<p>카테고리와 태그 페이지를 만들었는데, 주소창에 입력을 통해서만 들어갈 수 있어서 불편하다. 상단 메뉴를 수정하고 카테고리 페이지를 추가해 볼 것이다. 상단 메뉴 <code>Quick-Start Guide</code>가 있는 곳이다.</p>
<p>메뉴에 관한 설정은 <code>/_data/navigation.yml</code> 파일에 들어있다.</p>
<pre><code class="language-yml"># main links
main:
  - title: 'Quick-Start Guide'
    url: https://mmistakes.github.io/minimal-mistakes/docs/quick-start-guide/
  # - title: "About"
  #   url: https://mmistakes.github.io/minimal-mistakes/about/
  # - title: "Sample Posts"
  #   url: /year-archive/
  # - title: "Sample Collections"
  #   url: /collection-archive/
  # - title: "Sitemap"
  #   url: /sitemap/
</code></pre>
<p><code>title</code>에는 메뉴 이름을, <code>url</code>에는 링크 주소를 적으면 된다.</p>
<p>Quick-Start Guide는 지우고, 홈으로 가는 메뉴와 내가 만든 카테고리 페이지를 추가해보겠다.</p>
<pre><code class="language-yml"># main links
main:
  - title: 'Home'
    url: /
  - title: 'Categories'
    url: /categories/
</code></pre>
<p>주소 전체를 입력하지 않으면 내 블로그를 기준으로 링크가 생성된다.</p>
<p>이제 <code>Home</code>과 <code>Categories</code> 메뉴가 생겼다. 클릭하면 해당 페이지로 이동한다.</p>
