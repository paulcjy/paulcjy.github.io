b:Tf3e,<h1>1. Github에서 Repository 생성하기</h1>
<p>Github 블로그는 repository를 이용해서 관리한다. 주의할 점은, repository 이름이 <em>username</em>.github.io일 때에만 블로그로 사용이 가능하다. <em>username</em>에는 본인의 계정 이름을 넣으면 된다. 그러고 여기에 내용물이 되는 파일들을 업로드한다.</p>
<p><strong>New를 클릭하여 새 repository 생성</strong></p>
<p><img src="https://user-images.githubusercontent.com/86853786/128949181-d862f456-7e8d-4108-8e9c-5f0f2e419a7e.png" alt="image"></p>
<p><strong>repository 이름을 계정 이름과 같게 입력</strong></p>
<h1>2. Repository Clone 하기</h1>
<p>Github 홈페이지에서 파일을 작성하고 수정하는 방법으로도 블로그를 만들 수 있겠지만, 편하게 작성하기 위해 컴퓨터로 repository를 복사해 온다. 이렇게 하면 원래 사용하던 코드 에디터로 관리할 수 있어서 편하다. Github repository에 있는 파일들이 그대로 내 컴퓨터에도 저장되고, <code>git</code>을 이용하여 두 곳을 연동한다. 그러기 위해서는 먼저 <code>git</code>이 설치되어 있어야 한다.</p>
<h2>만약 git이 없다면</h2>
<p><a href="https://git-scm.com/">https://git-scm.com/</a>에서 다운로드 받을 수 있다.</p>
<h2>Repository 주소 복사하기</h2>
<p>Clone 하기 위해서는 Github repository의 주소가 있어야 한다. 주소는 새로 만든 repository의 주소에 <code>.git</code>을 붙인 것이다.</p>
<pre><code>https://github.com/username/username.github.io.git
</code></pre>
<p>이 주소는 Github에서 쉽게 복사할 수 있다. 새로 만든 repository에 아무 파일이 없다면 이런 화면이 뜰 것이다.</p>
<p><img src="https://user-images.githubusercontent.com/86853786/128952319-04bac245-67c8-409c-a679-be4ddb904285.png" alt="image"></p>
<p>맨 위에 있는 주소가 repository의 주소이다.</p>
<p>repository에 파일이 있다면 초록색 <code>Code</code> 버튼을 누르면 된다. 주소 오른쪽의 복사하기 버튼을 눌러 복사한다.</p>
<p><img src="https://user-images.githubusercontent.com/86853786/128951285-ac3cfc44-680e-4680-98dc-20a229743681.png" alt="image"></p>
<h2>명령어로 Clone 하기</h2>
<p>이제 명령 프롬프트를 열고, repository를 저장하고 싶은 디렉토리로 이동한다. 디렉토리 이동은 <code>cd</code> 명령어로 할 수 있다. 예를 들어, C:\blog 위치에서 작업하고 싶다면 <code>cd C:\blog</code>를 입력하고 엔터를 누르면 된다.</p>
<p>이동했다면, <code>git clone</code> 명령으로 repository를 받아온다.</p>
<pre><code>get clone https://github.com/username/username.github.io.git
</code></pre>
<p>이제 명령 프롬프트에서 이동했던 폴더로 가면 <code>username.github.io</code>라는 폴더가 생겼을 것이다. 이 폴더와 Github의 repository는 원격으로 연동되어 있다.</p>
<h1>3. 페이지 만들어 업로드하기</h1>
<p>간단하게 웹 페이지가 잘 뜨는지만 확인하기 위해 폴더에 index.html을 만들어 보자. 내용은 hello world 같이 아무 내용이나 적으면 된다.</p>
<p>다시 명령 프롬프트에서 clone 해온 폴더로 이동한다. 예시처럼 <code>C:\blog</code>에서 clone 했다면 <code>C:\blog\username.github.io</code>로 이동한다.</p>
<p>그리고 다음 명령어 3개를 입력한다.</p>
<pre><code class="language-sh">> git add .
> git commit -m "남기고 싶은 메시지"
> git push origin main
</code></pre>
<p>이 명령들은 현재 내 컴퓨터에 있는 폴더를 그대로 Github 홈페이지에 적용하겠다는 명령들이다.</p>
<p>이제 웹 브라우저를 켜고 본인의 주소를 치면 작성했던 index.html이 브라우저에 나타날 것이다.</p>
