Github Pages로 블로그 만들기 (9) - 카테고리 메뉴 만들기

2021. 08. 11.

카테고리와 태그 페이지를 만들었는데, 주소창에 입력을 통해서만 들어갈 수 있어서 불편하다. 상단 메뉴를 수정하고 카테고리 페이지를 추가해 볼 것이다. 상단 메뉴 Quick-Start Guide가 있는 곳이다.

메뉴에 관한 설정은 /_data/navigation.yml 파일에 들어있다.

# main links
main:
  - title: 'Quick-Start Guide'
    url: https://mmistakes.github.io/minimal-mistakes/docs/quick-start-guide/
  # - title: "About"
  #   url: https://mmistakes.github.io/minimal-mistakes/about/
  # - title: "Sample Posts"
  #   url: /year-archive/
  # - title: "Sample Collections"
  #   url: /collection-archive/
  # - title: "Sitemap"
  #   url: /sitemap/

title에는 메뉴 이름을, url에는 링크 주소를 적으면 된다.

Quick-Start Guide는 지우고, 홈으로 가는 메뉴와 내가 만든 카테고리 페이지를 추가해보겠다.

# main links
main:
  - title: 'Home'
    url: /
  - title: 'Categories'
    url: /categories/

주소 전체를 입력하지 않으면 내 블로그를 기준으로 링크가 생성된다.

이제 Home과 Categories 메뉴가 생겼다. 클릭하면 해당 페이지로 이동한다.
