# 시간복잡도

| n          | O            |
| ---------- | ------------ |
| 11         | O(n!)        |
| 25         | O(2^n)       |
| 100        | O(n^4)       |
| 500        | O(n^3)       |
| 3,000      | O(n^2 log n) |
| 5,000      | O(n^2)       |
| 1,000,000  | O(n log n)   |
| 10,000,000 | O(n)         |
| ~          | O(log n)     |
